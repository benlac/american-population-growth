# US population growth visualization app 

An American Population Growth app made with ReactJS and React-Leaflet

## Installation
To setup the project, install the dependencies
```bash
yarn
```
Compiles and hot-reloads for development
```bash
yarn start
```

## Technology used

- Javascript
- React.js
- Webpack and Babel
- Sass
